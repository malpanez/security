Create `roles/tls_hardening/tasks/enforce.yml` with the following structure. CRITICAL: all tasks tagged with [tls_hardening, enforce] plus section-level tags (configure, verify).

**Section 1: FIPS detection (all platforms)**

1. **Enforce | Read FIPS state** — `ansible.builtin.slurp` src `/proc/sys/crypto/fips_enabled`, register `_tls_fips_raw`, `failed_when: false`, tags: [tls_hardening, enforce, configure]

2. **Enforce | Set FIPS fact** — `ansible.builtin.set_fact` `_tls_fips_active: "{{ (_tls_fips_raw.content | default('MA==') | b64decode | trim) == '1' }}"`, tags: [tls_hardening, enforce, configure]. Note: MA== is base64 for '0'.

3. **Enforce | Skip enforcement when FIPS active** — `ansible.builtin.debug` msg "FIPS mode is active - skipping TLS hardening enforcement (FIPS policy supersedes)", when: `_tls_fips_active | bool`, tags: [tls_hardening, enforce, configure] (per D-03)

**Section 2: RHEL 8/9 crypto-policies branch**

All tasks in this section have when conditions including: `ansible_os_family == 'RedHat'`, `ansible_distribution_major_version | int >= 8`, `not _tls_fips_active | bool`

4. **Enforce | Install crypto-policies-scripts (RHEL 8/9)** — `ansible.builtin.package` name `crypto-policies-scripts` state `present`, when: RHEL 8/9 + not FIPS, tags: [tls_hardening, enforce, install]

5. **Enforce | Read current crypto policy** — `ansible.builtin.command` cmd `update-crypto-policies --show`, register `_tls_current_policy`, `changed_when: false`, `failed_when: false`, when: RHEL 8/9 + not FIPS, tags: [tls_hardening, enforce, configure]

6. **Enforce | Set crypto policy** — `ansible.builtin.command` cmd `update-crypto-policies --set {{ tls_hardening_rhel_policy }}`, `changed_when: true`, when: RHEL 8/9 + not FIPS + `(_tls_current_policy.stdout | default('')) != tls_hardening_rhel_policy` (per D-04 — idempotent: only fires when policy differs), tags: [tls_hardening, enforce, configure]

**Section 3: openssl.cnf branch (Debian, SUSE, RHEL<8)**

All tasks in this section have when: `(ansible_os_family in ['Debian', 'Suse'] or (ansible_os_family == 'RedHat' and ansible_distribution_major_version | int < 8)) and not _tls_fips_active | bool`. Use a YAML anchor or define a set_fact `_tls_use_openssl_cnf` to reduce repetition.

7. **Enforce | Set openssl.cnf branch fact** — `ansible.builtin.set_fact` `_tls_use_openssl_cnf: "{{ ansible_os_family in ['Debian', 'Suse'] or (ansible_os_family == 'RedHat' and ansible_distribution_major_version | int < 8) }}"`, tags: [tls_hardening, enforce, configure]

8. **Enforce | Slurp openssl.cnf** — `ansible.builtin.slurp` src `{{ tls_hardening_openssl_cnf }}`, register `_tls_cnf_content`, `failed_when: false`, when: `_tls_use_openssl_cnf | bool and not _tls_fips_active | bool`, tags: [tls_hardening, enforce, configure]

9. **Enforce | Check if system_default_sect exists** — `ansible.builtin.set_fact` `_tls_sect_absent: "{{ _tls_cnf_content.content is not defined or '[system_default_sect]' not in (_tls_cnf_content.content | b64decode) }}"`, when: `_tls_use_openssl_cnf | bool and not _tls_fips_active | bool`, tags: [tls_hardening, enforce, configure]

10. **Enforce | Add ssl_conf linkage to openssl_init** — `ansible.builtin.lineinfile` path `{{ tls_hardening_openssl_cnf }}`, regexp `^\s*ssl_conf\s*=`, line `ssl_conf = ssl_sect`, insertafter `\[openssl_init\]`, backup true. when: `_tls_use_openssl_cnf | bool and not _tls_fips_active | bool and _tls_sect_absent | bool`. tags: [tls_hardening, enforce, configure]. CRITICAL: this is the chain linkage that makes [system_default_sect] effective on OpenSSL 3.x (Pitfall 1 from RESEARCH).

11. **Enforce | Insert TLS policy block** — `ansible.builtin.blockinfile` path `{{ tls_hardening_openssl_cnf }}`, marker `# {mark} ANSIBLE MANAGED BLOCK tls_hardening`, insertafter `EOF`, block content (exactly):
    ```
    [ssl_sect]
    system_default = system_default_sect

    [system_default_sect]
    MinProtocol = {{ tls_hardening_min_protocol }}
    CipherString = {{ tls_hardening_openssl_cipher_string }}
    Options = ServerPreference
    ```
    backup true, when: `_tls_use_openssl_cnf | bool and not _tls_fips_active | bool and _tls_sect_absent | bool`, tags: [tls_hardening, enforce, configure]

12. **Enforce | Ensure MinProtocol value** — `ansible.builtin.lineinfile` path `{{ tls_hardening_openssl_cnf }}`, regexp `^\s*MinProtocol\s*=`, line `MinProtocol = {{ tls_hardening_min_protocol }}`, insertafter `\[system_default_sect\]`, backup true, when: `_tls_use_openssl_cnf | bool and not _tls_fips_active | bool and not _tls_sect_absent | bool`, tags: [tls_hardening, enforce, configure]

13. **Enforce | Ensure CipherString value** — `ansible.builtin.lineinfile` path `{{ tls_hardening_openssl_cnf }}`, regexp `^\s*CipherString\s*=`, line `CipherString = {{ tls_hardening_openssl_cipher_string }}`, insertafter `\[system_default_sect\]`, backup true, when: `_tls_use_openssl_cnf | bool and not _tls_fips_active | bool and not _tls_sect_absent | bool`, tags: [tls_hardening, enforce, configure]

**Section 4: Completion report**
