```yaml
---
- name: Gather minimum facts if not already collected
  ansible.builtin.setup:
    gather_subset: [min]
  when: ansible_facts == {}
  tags: [always]

- name: Assert supported OS family
  ansible.builtin.assert:
    that:
      - ansible_os_family in ['Debian', 'RedHat', 'Suse']
    fail_msg: >-
      antivirus: unsupported OS family '{{ ansible_os_family }}'.
      Supported: Debian, RedHat, Suse.
    quiet: true
  tags: [always]

- name: Load OS-specific variables
  ansible.builtin.include_vars: "{{ ansible_os_family }}.yml"
  tags: [always]

- name: Detect container environment
  ansible.builtin.set_fact:
    _antivirus_in_container: >-
      {{ ansible_facts.virtualization_type | default('') in
         ['docker', 'podman', 'container', 'lxc'] }}
  tags: [always]

- name: Review antivirus configuration
  ansible.builtin.include_tasks: review.yml
  tags: [antivirus, review]

- name: Enforce antivirus hardening
  ansible.builtin.include_tasks: enforce.yml
  when:
    - antivirus_enabled | bool
    - security_mode | default('review') == 'enforce'
  tags: [antivirus, enforce]
```

Note: `security_mode | default('review')` — NEVER `default('enforce')` per project rules.

Create `roles/antivirus/tasks/enforce.yml` with tagged sections (STD-04). All tasks MUST use FQCN (STD-05).

**Install section** (tags: [antivirus, enforce, install]):
1. **Install EPEL** — `ansible.builtin.dnf` with `name: "https://dl.fedoraproject.org/pub/epel/epel-release-latest-{{ ansible_distribution_major_version }}.noarch.rpm"`, `state: present`, `when: ansible_os_family == 'RedHat' and antivirus_epel_required | bool`
2. **Install ClamAV (RedHat)** — `ansible.builtin.dnf name: "{{ antivirus_packages }}" state: present`, `when: ansible_os_family == 'RedHat'`
3. **Install ClamAV (Debian)** — `ansible.builtin.apt name: "{{ antivirus_packages }}" state: present cache_valid_time: 3600`, `when: ansible_os_family == 'Debian'`
4. **Install ClamAV (Suse)** — `community.general.zypper name: "{{ antivirus_packages }}" state: present`, `when: ansible_os_family == 'Suse'`

**Configure section** (tags: [antivirus, enforce, configure]):
5. **Create log directory** — `ansible.builtin.file path: "{{ antivirus_log_dir }}" state: directory owner: "{{ antivirus_clamd_user }}" group: "{{ antivirus_clamd_group }}" mode: "0750"`
6. **Create scan log directory** — `ansible.builtin.file path: "{{ antivirus_scan_log | dirname }}" state: directory owner: "{{ antivirus_clamd_user }}" group: "{{ antivirus_clamd_group }}" mode: "0750"`
7. **Ensure DB directory ownership** — `ansible.builtin.file path: "{{ antivirus_db_dir }}" state: directory owner: "{{ antivirus_clamd_user }}" group: "{{ antivirus_clamd_group }}" mode: "0755"`
8. **Deploy clamd.conf** — `ansible.builtin.template src: clamd.conf.j2 dest: "{{ antivirus_clamd_config }}" owner: root group: root mode: "0644" backup: true`, `notify: Restart clamd`
9. **Deploy freshclam.conf** — `ansible.builtin.template src: freshclam.conf.j2 dest: "{{ antivirus_freshclam_config }}" owner: root group: root mode: "0644" backup: true`, `notify: Restart freshclam`
10. **SELinux boolean** — `ansible.posix.seboolean name: antivirus_can_scan_system state: true persistent: true`, `when: ansible_os_family == 'RedHat' and antivirus_selinux_contexts | bool` (research-identified gap — NOT in ROADMAP but REQUIRED on RHEL 9)
11. **Restore SELinux context on log directory** — `ansible.builtin.command cmd: "restorecon -Rv {{ antivirus_log_dir }}"`, register `_antivirus_restorecon_log`, `changed_when: "'Relabeled' in _antivirus_restorecon_log.stdout"`, `failed_when: false`, `when: ansible_os_family == 'RedHat' and antivirus_selinux_contexts | bool`
12. **Restore SELinux context on scan dirs** — `ansible.builtin.command cmd: "restorecon -Rv {{ item }}"`, `loop: "{{ antivirus_scan_dirs }}"`, register `_antivirus_restorecon_scan`, `changed_when: "'Relabeled' in _antivirus_restorecon_scan.stdout"`, `failed_when: false`, `when: ansible_os_family == 'RedHat' and antivirus_selinux_contexts | bool`
13. **Run freshclam** — `ansible.builtin.command cmd: freshclam`, register `_antivirus_freshclam_result`, `changed_when: "'Database updated' in _antivirus_freshclam_result.stdout"`, `failed_when: _antivirus_freshclam_result.rc not in [0, 1]`, `when: antivirus_update_db | bool and not _antivirus_in_container | bool` (AV-05 + AV-06)

**Service section** (tags: [antivirus, enforce, service]):
14. **Enable and start freshclam** — `ansible.builtin.systemd name: "{{ antivirus_freshclam_service }}" enabled: true state: started daemon_reload: true`, `when: not _antivirus_in_container | bool`
15. **Enable and start clamd** — `ansible.builtin.systemd name: "{{ antivirus_clamd_service }}" enabled: true state: started daemon_reload: true`, `when: not _antivirus_in_container | bool`
16. **Deploy clamav-scan.service** — `ansible.builtin.template src: clamav-scan.service.j2 dest: /etc/systemd/system/clamav-scan.service owner: root group: root mode: "0644"`, `notify: Reload systemd`, `when: antivirus_scan_enabled | bool`
17. **Deploy clamav-scan.timer** — `ansible.builtin.template src: clamav-scan.timer.j2 dest: /etc/systemd/system/clamav-scan.timer owner: root group: root mode: "0644"`, `notify: Reload systemd`, `when: antivirus_scan_enabled | bool`
18. **Enable and start scan timer** — `ansible.builtin.systemd name: clamav-scan.timer enabled: true state: started daemon_reload: true`, `when: antivirus_scan_enabled | bool and not _antivirus_in_container | bool`

**Verify section** (tags: [antivirus, enforce, verify]):
19. **Assert clamd config deployed** — `ansible.builtin.stat path: "{{ antivirus_clamd_config }}"` register, then `ansible.builtin.assert that: _result.stat.exists fail_msg: "clamd config not found at {{ antivirus_clamd_config }}"`
20. **Warn if DB absent and update_db false** — `ansible.builtin.debug msg: "WARNING: antivirus_update_db is false — virus database may not exist. Set antivirus_update_db: true for production use."`, `when: not antivirus_update_db | bool`
