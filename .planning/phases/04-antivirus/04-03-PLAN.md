1. Create `roles/antivirus/templates/clamd.conf.j2`:
```
# {{ ansible_managed }}
LogFile {{ antivirus_log_dir }}/clamd.log
LogTime yes
LogRotate yes
PidFile /run/clamd.scan/clamd.pid
TemporaryDirectory /var/tmp
DatabaseDirectory {{ antivirus_db_dir }}
LocalSocket /run/clamd.scan/clamd.sock
User {{ antivirus_clamd_user }}
MaxFileSize {{ antivirus_max_file_size }}
MaxScanSize {{ antivirus_max_scan_size }}
ScanPE yes
ScanELF yes
ScanOLE2 yes
ScanMail yes
ScanArchive yes
DetectPUA yes
FollowFileSymlinks false
FollowDirectorySymlinks false
```

2. Create `roles/antivirus/templates/freshclam.conf.j2`:
```
# {{ ansible_managed }}
DatabaseOwner {{ antivirus_clamd_user }}
UpdateLogFile {{ antivirus_log_dir }}/freshclam.log
LogRotate yes
LogTime yes
DatabaseDirectory {{ antivirus_db_dir }}
DatabaseMirror database.clamav.net
Checks {{ antivirus_freshclam_checks }}
NotifyClamd {{ antivirus_clamd_config }}
```
CRITICAL: `NotifyClamd` takes the path to the clamd config file (not socket). freshclam reads that file to find the socket path. Use `{{ antivirus_clamd_config }}` which resolves to `/etc/clamd.d/scan.conf` on RHEL, `/etc/clamav/clamd.conf` on Debian, etc.

3. Create `roles/antivirus/templates/clamav-scan.service.j2`:
```
# {{ ansible_managed }}
[Unit]
Description=ClamAV virus scan
Documentation=man:clamscan(1)
After=network.target
ConditionPathExists=/usr/bin/clamscan

[Service]
Type=oneshot
User=root
ExecStart=/usr/bin/clamscan \
  --recursive \
  --infected \
  --log={{ antivirus_scan_log }} \
{% for dir in antivirus_scan_dirs %}
  {{ dir }}{{ '' if loop.last else ' \\' }}
{% endfor %}
```

4. Create `roles/antivirus/templates/clamav-scan.timer.j2`:
```
# {{ ansible_managed }}
[Unit]
Description=ClamAV periodic virus scan

[Timer]
OnCalendar={{ antivirus_scan_schedule }}
Persistent=true
RandomizedDelaySec=300

[Install]
WantedBy=timers.target
```
`Persistent=true` ensures the timer fires after a missed window. `RandomizedDelaySec=300` avoids thundering-herd.
