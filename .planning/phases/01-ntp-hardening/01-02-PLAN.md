```yaml
  tags: [always]

  tags: [always]

- name: Detect container environment
  ansible.builtin.set_fact:
    _ntp_in_container: >-
      {{ ansible_facts.virtualization_type | default('') in
         ['docker', 'podman', 'container', 'lxc'] }}
  tags: [always]

- name: Review NTP configuration
  ansible.builtin.include_tasks: review.yml
  tags: [ntp_hardening, review]

- name: Enforce NTP hardening
  ansible.builtin.include_tasks: enforce.yml
  when:
    - ntp_hardening_enabled | bool
    - security_mode | default('review') == 'enforce'
  tags: [ntp_hardening, enforce]
```

Write tasks/review.yml with ONLY read-only tasks (STD-03). Every command task must have `changed_when: false`. Every probe must have `failed_when: false` (package may not be installed).

Tasks to include:

1. **Check chrony package** — use `ansible.builtin.command: rpm -q chrony` when `ansible_os_family in ['RedHat', 'Suse']`, and `ansible.builtin.command: dpkg-query -W chrony` when `ansible_os_family == 'Debian'`. Both with `changed_when: false`, `failed_when: false`. Register `_ntp_pkg_check`.

2. **Check chronyd service status** — `ansible.builtin.command: systemctl is-active {{ ntp_hardening_service }}` with `changed_when: false`, `failed_when: false`. Register `_ntp_svc_check`.

3. **Check chronyc tracking** — `ansible.builtin.command: chronyc tracking` with `changed_when: false`, `failed_when: false`, `when: not _ntp_in_container | bool`. Register `_ntp_tracking`.

4. **Check chronyc sources** — `ansible.builtin.command: chronyc sources -v` with `changed_when: false`, `failed_when: false`, `when: not _ntp_in_container | bool`. Register `_ntp_sources`.

5. **Check for CVE-2013-5211 risk (monitor directive)** — `ansible.builtin.command: grep -iE '^\s*(monitor|cmdallow)' {{ ntp_hardening_config_file }}` with `changed_when: false`, `failed_when: false`. Register `_ntp_monitor_check`. The check is: `rc == 0` means vulnerable (monitor directive found).

6. **Report NTP review findings** — `ansible.builtin.debug` with a `msg` that includes:
   - Package installed: `{{ _ntp_pkg_check.rc == 0 }}`
   - Service active: `{{ _ntp_svc_check.stdout | default('unknown') }}`
   - Tracking: `{{ _ntp_tracking.stdout_lines | default(['skipped - container']) }}`
   - Sources: `{{ _ntp_sources.stdout_lines | default(['skipped - container']) }}`
   - CVE-2013-5211 risk (monitor directive present): `{{ _ntp_monitor_check.rc == 0 }}`

All tasks tagged: `[ntp_hardening, review]`.

Write tasks/enforce.yml with four tagged sections (STD-04):

**[install] tag:**

1. Install chrony on RedHat:
```yaml
- name: Enforce | Install chrony (RedHat)
  ansible.builtin.dnf:
    name: "{{ ntp_hardening_package }}"
    state: present
  when: ansible_os_family == 'RedHat'
  tags: [install]
```

2. Install chrony on Debian:
```yaml
- name: Enforce | Install chrony (Debian)
  ansible.builtin.apt:
    name: "{{ ntp_hardening_package }}"
    state: present
    cache_valid_time: 3600
  when: ansible_os_family == 'Debian'
  tags: [install]
```

3. Install chrony on Suse:
```yaml
- name: Enforce | Install chrony (Suse)
  community.general.zypper:
    name: "{{ ntp_hardening_package }}"
    state: present
  when: ansible_os_family == 'Suse'
  tags: [install]
```

**[configure] tag:**

4. Disable systemd-timesyncd (Debian only):
```yaml
- name: Enforce | Disable systemd-timesyncd (Debian)
  ansible.builtin.systemd:
    name: systemd-timesyncd
    enabled: false
    state: stopped
  failed_when: false
  when: ansible_os_family == 'Debian'
  tags: [configure]
```

5. Disable legacy ntpd (RedHat only):
```yaml
- name: Enforce | Disable legacy ntpd (RedHat)
  ansible.builtin.systemd:
    name: ntpd
    enabled: false
    state: stopped
  failed_when: false
  when: ansible_os_family == 'RedHat'
  tags: [configure]
```

6. Create log directory:
```yaml
- name: Enforce | Create chrony log directory
  ansible.builtin.file:
    path: "{{ ntp_hardening_logdir }}"
    state: directory
    owner: root
    group: root
    mode: "0750"
  tags: [configure]
```

7. Deploy chrony.conf template:
```yaml
- name: Enforce | Deploy chrony.conf
  ansible.builtin.template:
    src: chrony.conf.j2
    dest: "{{ ntp_hardening_config_file }}"
    owner: root
    group: root
    mode: "0644"
    backup: true
  notify: Restart chronyd
  tags: [configure]
```

**[service] tag:**

8. Enable and start chronyd:
```yaml
- name: Enforce | Enable and start chronyd
  ansible.builtin.systemd:
    name: "{{ ntp_hardening_service }}"
    enabled: true
    state: started
    daemon_reload: true
  when: not _ntp_in_container | bool
  tags: [service]
```

**[verify] tag:**

9. Verify chronyc tracking:
```yaml
- name: Enforce | Verify chrony tracking
  ansible.builtin.command:
    cmd: chronyc tracking
  changed_when: false
  when: not _ntp_in_container | bool
  tags: [verify]
```
