**Template 1: `roles/rsyslog_forwarding/templates/99-forwarding.conf.j2`**

Write the template with exactly this structure (per D-15, D-16, LOG-02, LOG-03, LOG-04):

```jinja2
# Managed by Ansible — malpanez.security.rsyslog_forwarding
# Do not edit manually.

{% if rsyslog_forwarding_tls_enabled | bool %}
module(load="imtls")

global(
  defaultNetstreamDriver="gtls"
  defaultNetstreamDriverCAFile="{{ rsyslog_forwarding_tls_ca_cert }}"
{% if rsyslog_forwarding_tls_client_cert | default('') | length > 0 and rsyslog_forwarding_tls_client_key | default('') | length > 0 %}
  defaultNetstreamDriverCertFile="{{ rsyslog_forwarding_tls_client_cert }}"
  defaultNetstreamDriverKeyFile="{{ rsyslog_forwarding_tls_client_key }}"
{% endif %}
)

{% endif %}
$FileCreateMode {{ rsyslog_forwarding_file_create_mode }}

{{ rsyslog_forwarding_selector }} action(
  type="omfwd"
  Target="{{ rsyslog_forwarding_host }}"
  Port="{{ rsyslog_forwarding_port }}"
  Protocol="{{ rsyslog_forwarding_protocol }}"
{% if rsyslog_forwarding_tls_enabled | bool %}
  StreamDriver="gtls"
  StreamDriverMode="1"
  StreamDriverAuthMode="{{ rsyslog_forwarding_tls_auth_mode }}"
{% endif %}
  queue.type="{{ rsyslog_forwarding_queue_type }}"
  queue.size="{{ rsyslog_forwarding_queue_size }}"
{% if rsyslog_forwarding_queue_type == 'Disk' %}
  queue.spoolDirectory="{{ rsyslog_forwarding_queue_spool_dir }}"
  queue.filename="{{ rsyslog_forwarding_queue_file_name }}"
{% endif %}
  action.resumeRetryCount="{{ rsyslog_forwarding_resume_retry_count }}"
)
```

Key points:
- `module(load="imtls")` only inside TLS block (per D-16 RainerScript only)
- mTLS client cert/key only when BOTH variables are non-empty (per D-11)
- `queue.spoolDirectory` + `queue.filename` only when `queue_type == 'Disk'` (per D-05, Pitfall 6: queue.filename is a basename not full path)
- `$FileCreateMode` directive uses the variable value (per Claude's discretion)
- No legacy `@@host:port` syntax anywhere (per D-16)
- No `global()` block when TLS is disabled

**Template 2: `roles/rsyslog_forwarding/templates/rsyslog-logrotate.j2`**

Write the logrotate stanza (per D-07, D-09):

```jinja2
# Managed by Ansible — malpanez.security.rsyslog_forwarding
{{ rsyslog_forwarding_log_dir }}/*.log {
    rotate {{ rsyslog_forwarding_logrotate_rotate }}
    size {{ rsyslog_forwarding_logrotate_size }}
    missingok
    notifempty
    compress
    delaycompress
    sharedscripts
    postrotate
        /usr/bin/systemctl kill -s HUP rsyslog.service 2>/dev/null || true
    endscript
}
```

Key points:
- Deploys to `/etc/logrotate.d/rsyslog-forwarding` (dest is in enforce.yml task)
- Uses size-based rotation with `rsyslog_forwarding_logrotate_size` and `rsyslog_forwarding_logrotate_rotate`
- `compress` + `delaycompress` (per Claude's discretion)
- `sharedscripts` + HUP signal to rsyslog in postrotate
- `2>/dev/null || true` guards against missing service in containers
