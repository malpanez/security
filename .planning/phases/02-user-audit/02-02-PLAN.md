  <action>
Create `roles/user_audit/tasks/review.yml` with the following tasks in this exact order. Every task MUST have `changed_when: false` and `failed_when: false` (STD-03). All shell tasks MUST have `executable: /bin/bash` and `set -o pipefail &&` prefix. All task names start uppercase. All modules use FQCN (STD-05).

**CRITICAL: Use `lastlog --before` NOT `--time`. The --time flag returns RECENTLY ACTIVE accounts. The --before flag returns accounts INACTIVE longer than N days.**

1. **Collect human local accounts (UID >= 1000)**:
```yaml
- name: Review | Collect human local accounts (UID >= 1000)
  ansible.builtin.shell:
    cmd: >-
      set -o pipefail &&
      getent passwd |
      awk -F: '$3 >= 1000 && $7 !~ /nologin|false/ {print $1}'
    executable: /bin/bash
  register: _ua_human_accounts
  changed_when: false
  failed_when: false
  tags: [user_audit, review]
```

2. **Set human accounts list fact**:
```yaml
- name: Review | Set human accounts list fact
  ansible.builtin.set_fact:
    _ua_human_account_list: "{{ _ua_human_accounts.stdout_lines | default([]) }}"
  tags: [user_audit, review]
```

3. **Check password expiry for each human account**:
```yaml
- name: Review | Check password expiry for each human account
  ansible.builtin.command:
    cmd: chage -l {{ item }}
  loop: "{{ _ua_human_account_list }}"
  register: _ua_chage_results
  changed_when: false
  failed_when: false
  tags: [user_audit, review]
```

4. **Identify accounts with no password expiry** — filter on `Password expires.*never`:
```yaml
- name: Review | Identify accounts with no password expiry
  ansible.builtin.set_fact:
    _ua_no_expiry: >-
      {{
        _ua_chage_results.results | default([])
        | selectattr('stdout', 'defined')
        | selectattr('stdout', 'search', 'Password expires\s*:\s*never')
        | map(attribute='item')
        | list
      }}
  tags: [user_audit, review]
```

5. **Find UID 0 non-root accounts** (security check for duplicate UID 0):
```yaml
- name: Review | Find UID 0 non-root accounts
  ansible.builtin.shell:
    cmd: >-
      set -o pipefail &&
      awk -F: '$3 == 0 && $1 != "root" {print $1}' /etc/passwd
    executable: /bin/bash
  register: _ua_uid0_nonroot
  changed_when: false
  failed_when: false
  tags: [user_audit, review]
```

6. **Find accounts inactive longer than threshold** using `--before` (NOT `--time`):
```yaml
- name: Review | Find accounts inactive longer than threshold
  ansible.builtin.shell:
    cmd: >-
      set -o pipefail &&
      lastlog --before {{ user_audit_inactive_days }} |
      awk 'NR > 1 && $1 != "" && $1 != "Username" {print $1}'
    executable: /bin/bash
  register: _ua_inactive_by_days
  changed_when: false
  failed_when: false
  tags: [user_audit, review]
```

7. **Find service accounts with interactive login shells** (UID > 0, UID < 1000, shell is bash/sh/zsh):
```yaml
- name: Review | Find service accounts with interactive login shells
  ansible.builtin.shell:
    cmd: >-
      set -o pipefail &&
      awk -F: '$3 > 0 && $3 < 1000 && $7 ~ /bash|\/sh$|zsh/ {print $1}' /etc/passwd
    executable: /bin/bash
  register: _ua_service_login_shells
  changed_when: false
  failed_when: false
  tags: [user_audit, review]
```

8. **Check passwd/shadow consistency** using pwck (failed_when: false because pwck exits non-zero on Debian minimal — see pitfall 5 in RESEARCH):
```yaml
- name: Review | Check passwd/shadow consistency
  ansible.builtin.command:
    cmd: pwck -r -q
  register: _ua_pwck_result
  changed_when: false
  failed_when: false
  tags: [user_audit, review]
```

9. **Debug report with all findings**:
```yaml
- name: Review | User account audit report
  ansible.builtin.debug:
    msg:
      human_accounts: "{{ _ua_human_account_list }}"
      accounts_no_expiry: "{{ _ua_no_expiry | default([]) }}"
      uid0_nonroot: "{{ _ua_uid0_nonroot.stdout_lines | default([]) }}"
      inactive_accounts: >-
        {{
          _ua_inactive_by_days.stdout_lines | default([])
          | intersect(_ua_human_account_list)
        }}
      service_login_shells: "{{ _ua_service_login_shells.stdout_lines | default([]) }}"
      pwck_rc: "{{ _ua_pwck_result.rc | default('n/a') }}"
      pwck_output: "{{ _ua_pwck_result.stdout_lines | default([]) }}"
  tags: [user_audit, review]
```

The debug report for inactive_accounts MUST use `intersect(_ua_human_account_list)` to show only human accounts, not system accounts from lastlog --before output.
