Create `tasks/review.yml` with read-only tasks only (STD-03). Every command task MUST have `changed_when: false`. Every shell/command MUST have `failed_when: false` where the command may fail on systems without usbguard.

Task sequence:
1. `ansible.builtin.package_facts` with `manager: auto` — tags: [usbguard, review]
2. `ansible.builtin.set_fact` `_usbguard_installed: "{{ _usbguard_package in ansible_facts.packages }}"` — tags: [usbguard, review]
3. `ansible.builtin.command` `systemctl is-active {{ _usbguard_service }}` register `_usbguard_svc_active`, `changed_when: false`, `failed_when: false` — tags: [usbguard, review]. Only when `_usbguard_installed | bool` and `_usbguard_usb_available | bool`.
4. `ansible.builtin.command` `grep ImplicitPolicyTarget {{ _usbguard_daemon_conf }}` register `_usbguard_current_policy`, `changed_when: false`, `failed_when: false` — tags: [usbguard, review]. Only when `_usbguard_installed | bool`.
5. `ansible.builtin.stat` path `{{ _usbguard_initial_rules }}` register `_usbguard_rules_present` — tags: [usbguard, review]
6. `ansible.builtin.debug` block with `msg` containing a multi-line report (use `|` block scalar):
   - USB bus available: `{{ _usbguard_usb_available }}`
   - Package installed: `{{ _usbguard_installed }}`
   - Service state: `{{ _usbguard_svc_active.stdout | default('unknown') }}` (when USB available and installed)
   - Current ImplicitPolicyTarget: `{{ _usbguard_current_policy.stdout | default('not configured') }}`
   - Initial rules file present: `{{ _usbguard_rules_present.stat.exists | default(false) }}`
   - WARNING line when `not _usbguard_usb_available | bool`: "USB bus absent — container environment detected. Enforce will skip."
   - WARNING line when `_usbguard_needs_epel | default(false) | bool` and `ansible_os_family == 'RedHat'`: "RHEL 8 detected — EPEL may be required for usbguard package."
   Tags: [usbguard, review]

No ansible.builtin.service_facts needed — use direct systemctl command with failed_when:false for simplicity.
  </action>
