# Evidence Pack

Generated by the `compliance_evidence` role (configurable with `compliance_evidence_output_dir`, default: `/var/log/compliance`).

Includes:
- Configuration copies: `sshd_config`, `sudoers`, `audit_rules` (archived if a directory).
- Command output: `sestatus`, `pam_chain`, `pam_auth_update`, `sshd_effective_auth`, `sudoers_summary`, others defined in defaults.
- Named per host: `<host>-<artifact>`.
- Integrity manifest: `evidence.sha256` with SHA256 hashes.

Reproduction:
1. Set `compliance_evidence_enabled=true`.
2. Run a play that includes the `compliance_evidence` role.
3. Review artifacts in the configured directory.
